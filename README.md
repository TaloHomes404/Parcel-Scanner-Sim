# 📦 Parcel Scanner Sim

An Android application built with **Jetpack Compose** and the **MVVM architecture**, designed for small parcel shipping businesses and warehouse environments.

---

## 📖 Description

Parcel Scanner Sim is a mobile application that enables fast and convenient barcode scanning for parcels and shipping forms using **Google ML Kit**.

The app improves parcel handling processes by:
- reducing manual errors,
- accelerating workflow,
- improving data accuracy in warehouse operations.

---

## ✨ Features

- 📷 Real-time barcode scanning using CameraX
- 📦 Parcel and shipping form handling
- ⚡ Fast barcode processing powered by Google ML Kit
- 🧩 Clean and modern UI built with Jetpack Compose
- 🔄 Scalable MVVM architecture for maintainability and future expansion

---

## 🛠️ Technologies

- Kotlin
- Jetpack Compose
- MVVM Architecture
- ML Kit (Barcode Scanning)
- CameraX

## 🧪 QA & Testing

As part of the development process, I prepared and executed a structured QA approach to ensure application quality and stability.

### 📄 Test Documentation

- Created a **Test Plan** defining scope, objectives, and testing strategy.
- Designed **manual test cases** and test scenarios for core application functionalities.
- Prepared **API testing documentation** for backend communication validation.
- Defined testing coverage for AI-powered barcode scanning functionality.

### 🔍 Testing Activities

- Executed **manual functional tests** for all core features.
- Verified API endpoints using **Postman**.
- Performed performance testing using **Apache JMeter**.
- Monitored application performance using **Android Profiler** (CPU, memory, rendering).
- Tested barcode scanning accuracy under different lighting and environmental conditions.
- Validated error handling and edge-case scenarios during scanning operations.
- Verified overall stability and usability of the application.
